# TaylorMoDe-GS
Official PyTorch implementation of the IJCAI 2026 paper "TaylorMoDe-GS: Taylor-Driven Gaussian Splatting Motion Model for Multi-View Dynamic Scene Deblurring".

## 📢 News

* **[2026-05]** Our paper has been officially accepted to **IJCAI 2026**!  Project repository is created. The source code will be released soon!
